# BEST stateManager

StateManager for BEST modules.

## About

The BEST _stateManager_ ...

  * Manages state for each BEST Component

[See the `docs` folder](docs).

## Getting started

After cloning the repo ...

    $ npm install

## Tests

Run all of the tests with:

    $ npm test
